## Development

### Install

1. Make sure you are using the Python version specified in the `.python-version` file (or use [pyenv](https://github.com/pyenv/pyenv) for the automation)
2. Run `make install`

### Build

1. Run `make build`

### Test the Python code

1. Run `make py-test`
